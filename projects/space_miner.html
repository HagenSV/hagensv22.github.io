Space Miner (2022)

Languages/Tools: Java, Java Swing

Purpose: This project was the final project for AP computer science A. Goal was to create a program which utilizes skills that you learned in the class. I worked with a partner to create a basic game where you fly around in space mining astroids. The game featured multiple different materials, limited inventory space, and a space station where you could sell the materials and buy ship upgrades.
